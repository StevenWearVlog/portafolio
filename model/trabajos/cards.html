Cards

Este proyecto fue una exploración en la creación de tarjetas visuales (cards) desarrolladas directamente con código, en donde el objetivo principal era entender cómo las jerarquías, proporciones y tamaños pueden cambiar la manera en que percibimos la información. Trabajé con HTML y CSS para estructurar y dar estilo a cada tarjeta, cuidando el uso del color, la tipografía y la composición para generar armonía visual y claridad. Cada card fue diseñada con un propósito visual específico, desde destacar un elemento principal hasta organizar contenido de forma responsiva. Me enfoqué en cómo los elementos interactúan entre sí dentro del espacio limitado de una tarjeta, manteniendo la coherencia estética y visual. Este ejercicio me permitió afianzar mis conocimientos técnicos y, al mismo tiempo, aplicar principios de diseño visual en un entorno digital.
