Grid

Este proyecto fue un ejercicio centrado en la construcción de una retícula visual a partir de código, entendiendo su lógica, proporciones y estructura como base del diseño gráfico. A partir de una retícula previamente diseñada, desarrollé una réplica precisa utilizando HTML y CSS, prestando atención al espaciado, la alineación y la consistencia visual entre los módulos. Más allá de la ejecución técnica, el proyecto me permitió reforzar el valor del orden visual y la importancia de las estructuras invisibles que sostienen el diseño. Fue una forma de traducir principios tradicionales del diseño editorial al entorno digital, destacando cómo la retícula puede ser tanto una herramienta funcional como una guía estética. Este trabajo reafirmó mi interés por los sistemas gráficos y su potencial expresivo incluso en lo más estructurado.
