Empaque

El diseño de este empaque fue una invitación a construir una identidad visual con alma propia. El proyecto consistió en conceptualizar y diseñar el empaque para una cerveza artesanal, partiendo de una narrativa que combinara elementos naturales, gráficos y simbólicos. El reto fue traducir esa esencia en una etiqueta que conectara visualmente con el consumidor, equilibrando lo artesanal con lo contemporáneo. Trabajé desde el concepto hasta los acabados visuales, definiendo paleta de colores, tipografías y elementos gráficos que transmitieran una experiencia sensorial coherente con el producto. El resultado es una pieza que no solo protege, sino que comunica; que no solo se ve bien, sino que cuenta una historia. Este proyecto reforzó mi interés por el diseño como medio de expresión visual y como herramienta de conexión emocional.
